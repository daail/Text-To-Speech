🎙️ Text-to-Speech Converter
A lightweight, browser-based application that converts written text into natural-sounding speech. This project provides a seamless audio experience by leveraging built-in browser APIs, featuring a curated selection of high-quality Microsoft voices.

🚀 Features

Instant Synthesis: Converts text input to audio with zero latency.

Curated Voice Selection: Streamlined user experience featuring a refined list of 5 premium Microsoft voices (e.g., Microsoft David, Zira, Mark).

Playback Controls: Intuitive interface allowing users to generate and stop the speech output.

Responsive Design: Clean, accessible UI that scales perfectly across desktop and mobile devices.

🛠️ Tech Stack

Frontend: HTML5, CSS3

Scripting: JavaScript (ES6+)

Web API: Web Speech API (SpeechSynthesis)
